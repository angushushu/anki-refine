# Run and deploy your AI Studio app

View this app in [AI Studio](https://ai.studio/apps/drive/17AVN0XchzrndNPGdS94YB8jiIWYy8ske?fullscreenApplet=true)

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key (or use your Gemini API KEY/OpenAI SDK)
3. Run the app:
   `npm run dev`
